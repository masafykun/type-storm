---

## ✨ 特徴

- **単一HTML** — ビルド工程なし、`index.html` ひとつで完結
- **レスポンシブ** — PC・スマートフォン両対応
- **スコア記録** — ベストスコアを localStorage に保存
- **タイトル → プレイ → リザルト** — ゲームループが完結
- **外部依存なし** — CDN 以外のアセットを持たない自己完結構成
